## Deploy instructions

Deploy the contents of this folder as the site root. `index.html`, `_redirects` and all assets must remain together at the top level.

For the existing GitHub/Netlify workflow, copy these files over the current repository without deleting unrelated files, commit them, and allow Netlify to deploy the new commit. The public route remains:

`https://awakening-eden-library.netlify.app/start-here`
