# Awakening Eden Living Library SEO + Deployment Notes

Recommended Netlify site name: `awakening-eden-living-library`

Expected free Netlify URL if available:
`https://awakening-eden-living-library.netlify.app`

If Netlify says that name is already taken, choose one of these and then replace `https://awakening-eden-living-library.netlify.app` in `sitemap.xml`, `robots.txt`, and the canonical/OG tags inside the HTML files:

- `awakening-eden-library`
- `awakeningeden-living-library`
- `awakening-eden-regeneration-library`

SEO basics included in this package:

- Page titles and meta descriptions
- Canonical URLs
- Open Graph sharing tags
- Twitter card tags
- JSON-LD structured data
- `sitemap.xml`
- `robots.txt`
- `manifest.webmanifest`
- Netlify redirects for friendly paths like `/library`, `/films`, `/resources`, `/guides`, and `/start`

After deployment:

1. Open Netlify site settings and set the site name to `awakening-eden-living-library` if available.
2. Check that the homepage opens at `https://awakening-eden-living-library.netlify.app`.
3. Open `https://awakening-eden-living-library.netlify.app/sitemap.xml`.
4. Later, when you have a custom domain, replace the Netlify URL in all SEO files with the final domain.
